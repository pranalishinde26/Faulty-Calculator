# 🧮 Faulty Calculator

A simple JavaScript-based calculator that intentionally produces incorrect results 10% of the time.

This project demonstrates basic JavaScript concepts such as random number generation, user input handling, object mapping, and conditional logic.

---

## 🚀 How It Works

- The user enters:
  - First number
  - Operation (+, -, *, /)
  - Second number
- The program generates a random number using `Math.random()`.
- 90% of the time, it performs the correct calculation.
- 10% of the time, it intentionally swaps the operator and produces a wrong result.

## ▶️ How to Run

1. Download or clone the repository.
2. Open `index.html` in your browser.
3. Enter the requested values in the prompt dialogs.

No installation required.

## 🎯 Learning Objectives

- Understanding `Math.random()`
- Working with user input using `prompt()`
- Conditional logic using `if-else`
- Object-based operator mapping
- Using `eval()` for dynamic expression evaluation
